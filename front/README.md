# chess2pic-web (front end)
Quasar SPA for chess2pic web app.


## Prerequisites

 - Node JS v16+

Install node dependencies with
```bash
yarn
# or
npm install
```

## Building

```bash
quasar build
```

## Running

It is possible to run Quasar dev server with
```bash
quasar dev
```

But the preferred way is to run the [production server](../back/README.md).
